# 🎨 项目策划案：印象派绘画分类器

## 📌 项目基本信息

- **课程任务**：自主实现机器学习/深度学习算法，完成多分类任务，提交代码、报告、PPT。
- **选定数据集**：`Impressionist_Classifier_Data`（10类艺术家，每类约400张训练图、100张验证图）。
- **核心算法推荐**：**卷积神经网络（CNN）**（推荐从零构建一个经典的类 VGG 或轻量化 ResNet 架构，既满足“自主实现”的要求，又完美契合图像数据集）。
- **交付物创新**：以响应式滚动叙事网页（Scrollytelling Web-Report）作为最终成果的聚合媒介。

## 🧭 网页滚动叙事（Scrollytelling）视觉与内容动线设计

整个网页将采用**低饱和度、强调网格结构与大留白的极简现代主义风格**，让艺术图像与硬核代码、图表产生强烈的视觉碰撞。随着用户向下滚动鼠标，页面像一幅卷轴般层层递进。

### 📊 网页架构与滚动章节设计

| **章节（Chapter）**                              | **滚动视觉动效（Scroll Effect）**                            | **核心展示内容（Content）**                                  |
| ------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **01. 序幕：画布之上** (Hero & Intro)            | 随着滚动，10位画家的经典作品碎片在背景中交织、淡出，最终聚焦于项目标题。 | **项目背景**：介绍印象派的笔触与色彩特征，引出“如何让机器分辨艺术风格”的核心命题。 |
| **02. 第一幕：解析结构** (Data Exploration)      | 树状目录树（Dataset Tree）以动效形式逐层展开，右侧伴随各类别的样本缩略图。 | **数据洞察**：展示 10 个类别的数量分布（400张 Train / 100张 Val），分析图像尺寸、色彩直方图的异同。 |
| **03. 第二幕：画笔的数学** (Algorithm Principle) | 抽象的 CNN 神经网络层（卷积、池化、全连接）随着滚动，以三维或扁平几何体形式向前推进、展开。 | **算法原理**：图解自主实现的 CNN 架构。使用动态公式和前向传播动效，解释算子如何捕捉艺术品的“局部笔触”与“全局构图”。 |
| **04. 第三幕：机械织造** (Code Implementation)   | 页面左侧锁定核心网络架构代码，右侧随着滚动高亮解释 `nn.Module`、自建数据加载器（DataLoader）的实现细节。 | **纯手工代码**：不依赖高级封装，解构底层代码实现，证明“自主实现”的含金量。 |
| **05. 第四幕：理性的审判** (Results & Analysis)  | 曲线图实时绘制；滚动到特定区域时，**混淆矩阵**的单元格根据数值深浅渐变亮起。 | **实验分析**： 1. Loss 与 Accuracy 训练曲线。 2. **混淆矩阵**：深入分析哪些画家最容易被混淆（例如莫奈和毕沙罗）。 |
| **06. 🛠️ 独家亮点：机器之眼** (Interpretability)  | 滚动时，一张原作（如梵高的《星空》）在左侧，右侧渐变叠加 **Grad-CAM（类激活映射热力图）**。 | **模型可解释性**：展示网络在分类时究竟在看什么。机器是因为梵高的螺旋笔触认出他，还是因为莫奈的柔和色彩认出莫奈？ |
| **07. 尾声：画布之外** (Reflection)              | 背景噪声渐渐归于纯黑或纯白，大字重文字浮现。                 | **学习体会**：工程实践中的调参心得、艺术与人工智能的边界思考。 |

## 🛠️ 前端技术栈与开发建议

为了实现丝滑的滚动叙事效果，建议使用以下轻量且高效的工具：

- **滚动控制流**：使用 **GSAP (GreenSock) + ScrollTrigger**。这是目前最强大的滚动动画库，能精准控制滚动距离与动画帧的绑定。
- **数据可视化**：使用 **Chart.js** 或 **ApexCharts**，它们自带优雅的入场动画，非常适合在滚动到可视区域时触发图表生成。
- **代码高亮**：使用 **Prism.js** 或 **Highlight.js**，配合毛玻璃效果的容器。
